# 여성안심택배함 애플리케이션

## 설명
여성안심보관함의 위치를 Map상에서 확인하고 보관함에 맡겨진 나의 택배 현황을 알 수 있는 애플리케이션

## 개발목적
여성안심택배함의 위치를 한 눈에 볼 수 있는 애플리케이션이 없었기에,
사용자의 택배 물품을 관리하면서 택배함의 정보도 함께 조회할 수 있는 어플을 만들고자 함.
